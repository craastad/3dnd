# 3dnd

Our happy place to sell and buy used 3D printers.

Install NodeJs + NPM
https://docs.npmjs.com/getting-started/installing-node

Install dependencies (bootstrap)
`npm install`

Run build
`gulp`

Open the site locally :)
`dist/index.html`

Deploy By Running
`gulp deploy`

Open site on github
https://craastad.github.io/3dnd/
